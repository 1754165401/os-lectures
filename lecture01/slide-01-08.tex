\item 设计思路
    \begin{itemize}
    \item 采用小巧全面的操作系统ucore并进行改进，需要覆盖操作系统的关键点，为此增加：
    \begin{itemize}
        \item 外设：I/O管理/中断管理
        \item 内存：虚存管理/页表/缺页处理/页替换算法
        \item CPU：进程管理/调度器算法
        \item 并发：信号量实现和同步互斥应用
        \item 存储：文件系统+磁盘驱动
    \end{itemize}
    \item 完整代码量控制在10000行以内
    \item 提供实验讲义和源码分析文档
    \end{itemize}
\end{itemize}
